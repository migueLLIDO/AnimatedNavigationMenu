# 🧭 Animated Navigation Menu - Slider Interativo

Um menu de navegação moderno e interativo que utiliza um efeito de "slider" deslizante para indicar a opção selecionada. O design é focado na experiência do usuário (UX), proporcionando um feedback visual fluido e atraente.

## 🌟 Descrição

Este componente é um menu horizontal onde a seleção de itens é feita através de botões de rádio (radio buttons) ocultos, permitindo que o estado de "ativo" seja gerenciado puramente com CSS, sem a necessidade de JavaScript. 

Quando um item é clicado:
- Um indicador colorido (`slider`) desliza suavemente para a posição do item selecionado.
- O texto e o ícone do item ativo mudam de cor para branco, destacando-se sobre o fundo vibrante.
- O movimento do slider utiliza uma curva de animação `cubic-bezier`, proporcionando um efeito elástico e dinâmico.

## 🛠️ Funcionalidades Técnicas

- **Lógica Pure CSS:** Uso de seletores de irmãos (`~`) e pseudo-classes `:checked` para gerenciar o estado do menu.
- **Animações Fluidas:** Transição do slider com `cubic-bezier` para um movimento orgânico.
- **Layout Flexível:** Implementação com Flexbox para garantir o alinhamento perfeito dos itens.
- **Feedback Visual:** Efeito de hover suave nos itens não selecionados.
- **Ícones Integrados:** Uso de Font Awesome para representação visual intuitiva de cada categoria.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura lógica com inputs e labels.
- **CSS3:** Estilização avançada, animações de transição e sombras.
- **Font Awesome:** Ícones para as opções de navegação.
- **Google Fonts:** Fonte Poppins para um visual limpo e moderno.

## 📂 Como Usar

1. Baixe ou clone os arquivos do projeto.
2. Certifique-se de que o arquivo `css/style.css` está no caminho correto.
3. Abra o `index.html` em seu navegador preferido.
